# Webpage Generator Skill

## 概述

本技能用于根据用户需求自动生成高质量的网页代码，包括：
- 响应式布局设计
- 现代 CSS 动画效果
- 交互式组件
- 语义化 HTML 结构

## 使用方法

1. 描述你的网页需求（用途、风格、功能等）
2. 系统将自动生成完整的网页代码
3. 代码包含 HTML、CSS 和 JavaScript

## 特性

- **响应式设计**: 自动适配桌面、平板和手机
- **现代动画**: 使用 CSS 和 JavaScript 实现流畅动画
- **组件化**: 模块化组件结构，易于维护
- **最佳实践**: 遵循现代 Web 开发标准
